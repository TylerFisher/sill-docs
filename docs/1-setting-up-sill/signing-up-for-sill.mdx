# Create an account

You sign up for Sill by visiting [sill.social/accounts/signup](https://sill.social/accounts/signup). Then, follow these steps.

1. Enter your email address in the sign up form, and click "Sign up." Sill will send you an email to address you enter.

<div style={{
  maxWidth: "500px",
  margin: "0 auto"
}}>
<img src={require('./assets/signup.png').default} style={{
  width: "100%",
  height: "auto"
}} />
</div>

2. Go to your email inbox and copy the six-character code from the verification email. Paste it in the verification field on Sill. Click "Submit."

<div style={{
  maxWidth: "500px",
  margin: "0 auto"
}}>
<img src={require('./assets/verify.png').default} style={{
  width: "100%",
  height: "auto"
}} />
</div>

3. Enter your name and create a password for Sill. Make sure you save this password somewhere you'll remember it. Click "Create an account."

<div style={{
  maxWidth: "500px",
  margin: "0 auto"
}}>
<img src={require('./assets/onboarding.png').default} style={{
  width: "100%",
  height: "auto"
}} />
</div>

Next, you can [connect your social accounts to Sill](./connecting-accounts.mdx).
